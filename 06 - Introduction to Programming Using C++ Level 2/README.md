# 💻 06 - Introduction to Programming Using C++ – Level 2

> **"C++ becomes a power tool when you understand exactly why your code works."** 🧠

📌 This repository documents the sixth stage of my journey. After mastering problem-solving and clean code, I dived deep into the advanced mechanics of C++. This level is about **Control, Visibility, and Mastery** over memory and execution flow.

## 🗝️ About This Level (The Power of Control)
At this stage, I moved beyond basic syntax to understand the "under the hood" operations of C++. I focused on topics that separate hobbyists from professionals: **Debugging, Pointers, Memory Management, and File Handling.**

### 1️⃣ Professional Debugging & Execution
* **Mastering the Debugger:** Learned to use breakpoints and watches to inspect memory values and execution flow step-by-step.
* **Visibility:** Stopped "guessing" why code works and started "understanding" every line's impact on the system.

### 2️⃣ Memory & Pointer Mastery
* **Pointers & References:** Gained full confidence in using pointers and references to manipulate data directly in memory.
* **Dynamic Memory:** Learned to manage the **Stack vs. Heap** using `new` and `delete` for dynamic allocation.
* **Variable Lifetime:** Deeply understood storage duration and the scope of variables within complex programs.

### 3️⃣ Advanced Functions & Data Structures
* **Function Power:** Mastered advanced concepts like Function Overloading, Default Parameters, and **Recursion**.
* **Modern Structures:** Transitioned from basic arrays to using **Vectors** and Iterators for more flexible data management.
* **Custom Libraries:** Started creating and using my own C++ libraries to build reusable components.

### 4️⃣ Real-World Data Handling
* **File Streams:** Learned to read, write, update, and delete data using C++ File Handling.
* **Error Management:** Implemented **Exception Handling** to build reliable and crash-resistant applications.
* **Utilities:** Worked with Date, Time, and advanced string manipulation to handle real-world data types.
